Practicando uso de comandos de GitHub:

(Pagina oficial de GitHub)
  • Verifircar llaves SSH
    • Posicionarnos en la carpeta donde guardaremos las Keys
    • Abrir terminal y escribir las siguientes instrucciones
    • Busacar Key: ls -al ~/.ssh
    • Generar Key: $ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
    • Copiar la key generada a GitHub y listo!
  • Crear nuevo Repositorio y Subir codigo a dicho repo
    • Crear nuevo repositorio
    • Importar nuestro poryecto de la PC a GitHub
    • En la consola: git remote add origin https://github.com/alobp/Hola-Mundo-Git.git
    • Verificar con: git remote -v
    • Subir archivos: git push -u origin master
    • Guardar cambios: git push
    • Descargar archivos: git pull origin master
