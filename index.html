Practicando uso de comandos de Git

  • git init
  • git status
  • git add [archivo]
  • git add .
    • sirve para agregar todos los archivos al StaginArea
  • git restore [archivo]
  • git commit -m "comentario del commit"
  • git rm --cached [archivo]
    • sirve para quitar un archivo subido del StaginArea
  • git log
    • Nos muestra todos los 'commit' y sus comentarios asociados
  • git log --stat
    • Muestra los commit con info extensa
  • git log --oneline
    • Muestra solo los comentarios de los commit
  • git log --graph
    • Muestra una grafica (arbol) de los commit
  • git log --graph --oneline --all
    • Muestra una grafica de todos los comentarios de los commit
  • git diff
    • Muestra todos los cambios realizados (agregaciones en verde y eliminaciones en rojo), comparando el Archivo no guardado del ya guardado en git
  • git diff [id-commit] [archivo que queremos comparar]
    • Muestra todos los cambios realizados (agregaciones en verde y eliminaciones en rojo), comparando el Archivo del comit con el archivo especificado
  • git checkout [id-commit]
    • Restaura el archivo al estado que hayamos elegido en el ID-commit
  • git branch
    • Muestra las ramas de nuestro proyecto
  • git checkout -b [Nombre Rama]
    • Crea una nueva rama
  • git checkout [Nombre Rama]
    • Cambia a la rama seeccionada
